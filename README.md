# RAG Project

This project is a simple Retrieval-Augmented Generation (RAG) implementation.

## Folder Structure

```
.
├── data/
├── rag_db/
├── src/
│   ├── ingest.py
│   ├── rag.py
│   ├── run.py
│   └── tools.py
├── frankenstein.txt
├── requirements.txt
└── venv/
```

## Setup and Run

1.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the project:**
    ```bash
    python src/run.py
    ```
